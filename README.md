## 機能
### 機能リスト
 - 通常の計算
 - 文字を含む計算
 - 微分積分
 - 三角比
 - 平方根
 - 小数分数変換
 - 累乗
 - 最小公倍数
 - 最大公約数
 - データ
   - 平均値
   - 四分位数
   - 分散
   - 標準偏差
 - 関数グラフ
 - 計算練習
 - 辞書
 - 翻訳
